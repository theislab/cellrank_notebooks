CellRank notebooks
==================
**CellRank** is a modular framework to study cellular dynamics based on Markov state modeling of
multi-view single-cell data. Please see the `main repo`_ to learn more.

Running notebooks locally
-------------------------

Install `pixi <https://pixi.sh>`_, then:

.. code-block:: bash

    pixi install
    pixi run install-kernel
    pixi run lab
